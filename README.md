# Untitled UI

## 💻 Project

This project is a web application to apply TailwindCSS concepts using nextJS

## 🚀 Technologies

This project was developed with the following technologies:

- Tailwind CSS
- Radix UI
- Next
- Typescript
- Tailwind Merge
- Tailwind Variants
- Framer-Motion
- Responsive
- Dark Theme
- Label trigger all inputs
- Animations


## 🖲️ Getting Started
```sh
 - npm install
 - npm run dev
```
